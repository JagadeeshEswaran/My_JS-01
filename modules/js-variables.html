Declaration - Assignment - Re-Assignment
Declaration
  • var name1;
  • let name2;
  • const name3; // Error : "const" have to be initialized during declaration itself .
Assignment
  • var name1 = "Jagadeeshwaran"
  • let name2 = "Jagadeeshwaran" // Error : "name2" already declared, try to assign the variable without "let" keyword .
  • const name3 = "Jagadeeshwaran" // Will work properly now.
Re-Assignment
  • var name1 = "My Name" // Allowed, will work fine
  • let name2 = "My Name" // Not Allowed, try just using name2 = "My Name"
  • cpnst name3 = "My Name" // Not Allowed, Syntax Error
